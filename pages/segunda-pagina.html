Variáveis


As famosas variáveis estão presentes em todos os códigos de qualquer linguagem de programação. O intuito aqui não é explicar o que são variáveis, mas sim apresentar a melhor maneira de declará-las, de acordo com algumas regras e padrões.

Vejamos primeiro alguns exemplos de variáveis em um trecho de código em Java: 

            
                String nome = "João";
                String nomeMae = "Maria";
                int idade = 25;
                Date dataNascimento = new Date("15-01-1997");
                double salario = 1212.0;
                boolean possuiCarro = true;
Trecho de Código - 01

Acima, no Trecho de Código - 01, temos algumas variáveis de tipos diferentes, com elas podemos observar algumas características. Você consegue identificar quais são elas?
Primeiro, todas seguem uma regra chamada camelCase que determina que todas variáveis devem começar com letra minúscula e caso possua mais palavras na sua composição, a primeira letra de cada nova palavra deve ser maiúscula (como ocorre em nomeMae, dataNascimento e possuiCarro). Existem outras regras parecidas com essa, mas são usadas em cenários específicos, são elas: snake_case , PascalCase e kebab-case .

Outra regra para as variáveis é a de que elas não podem possuir acentos (isso pode estar relacionado ao fato de as linguagens de programação serem originárias de países que não utilizam acentos em suas palavras, como por exemplo nos países de língua inglesa).

No livro Código Limpo são apresentadas mais algumas recomendações quanto a nomenclatura de variáveis.
Não utilize abreviaturas, siglas, nomes que só você entende o significado, ou nomes que somente alguém com conhecimento profundo do sistema trabalhado conheça. Prefira nome completo que faça sentido!

Talvez você tenha aprendido, no seu curso de programação ou até mesmo na faculdade, que utilizar variáveis como a, b e c é a coisa mais linda do mundo. Mas acredite, não é e encontrá-las em um código é sempre uma dor de cabeça.

Para variáveis booleanas, procure escolher nomes que possam ser respondidos com Sim ou Não, como por exemplo a variável possuiCarro.

As constantes também são variáveis, mas geralmente utiliza-se um padrão para diferenciá-las das demais variáveis, elas são escritas com letras maíusculas e caso sejam compostas por mais de uma palavra utiliza-se a snake_case para separar cada palavra com um underline (exemplo de constante: ANO_ATUAL).
